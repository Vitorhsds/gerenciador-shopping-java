Gerenciador de Shopping Center

Projeto acadêmico desenvolvido em Java com o objetivo de simular o gerenciamento de estabelecimentos presentes em um Shopping Center.

Descrição

O sistema permite organizar e gerenciar diferentes segmentos de lojas encontrados em um shopping brasileiro, como:

Lojas de roupas
Lojas de eletrônicos
Cinema
Praça de alimentação
Lojas de calçados
Lojas de acessórios
Outros segmentos comerciais

A interface do projeto foi estruturada em formato de blocos (quadrados), representando visualmente os espaços ocupados pelas lojas dentro do shopping.

Funcionalidades
Cadastro de lojas
Listagem de lojas cadastradas
Organização por segmento
Visualização da estrutura do shopping
Simulação de gerenciamento de espaços comerciais

Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais da linguagem Java e da Programação Orientada a Objetos, simulando um cenário real de gerenciamento de lojas em um Shopping Center.
